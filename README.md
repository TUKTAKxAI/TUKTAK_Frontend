# TUKTAK_Frontend
This repository is for frontend developing in TUKTAK project.

## 프론트엔드 프로젝트 실행 방법

### 1. 저장소 복제

```bash
git clone 저장소_URL
```

`저장소_URL` 부분에는 실제 GitHub 저장소 주소를 입력합니다.

예시:

```bash
git clone https://github.com/조직명/tuktak-frontend.git
```

### 2. 프로젝트 폴더로 이동

```bash
cd tuktak-frontend
```

프로젝트 폴더명이 다른 경우 실제 폴더명으로 변경합니다.


### 3. 패키지 설치

```bash
npm install
```

`package.json`과 `package-lock.json`을 기준으로 프로젝트 실행에 필요한 패키지를 설치합니다.

### 4. 개발 서버 실행

```bash
npm run dev
```

실행 후 터미널에 출력되는 주소로 접속합니다.

기본 접속 주소:

```text
http://localhost:5173
```

### 전체 실행 명령어

```bash
git clone 저장소_URL
cd tuktak-frontend
npm install
npm run dev
```
